SisProKB

Mobile and Desktop Application for Keluarga Berencana Program Toba Samosir

SisProKB is integrated app between Desktop, Web, and Phone(Android) to manage the citizen in Toba Samosir,Sumatera Utara Region that participated in Keluarga Berencana (KB) program.
Android application input the data that will collected by the field officer and the data will be consumed by the desktop application.
The desktop Application will save the collected data and can be processed into reports
